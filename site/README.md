# Personal Blog (Astro + Axi theme base)

This site is prepared as an out-of-the-box personal blog for:

- Paper reading notes (`/blog/research`)
- Daily logs (`/blog/daily`)
- Project showcase (`/projects`)

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

See [HOW_TO_USE_CN.md](./HOW_TO_USE_CN.md) for Chinese usage instructions.
